### Step 2: Implement LLM Integration ‚ö†Ô∏è REQUIRED

```typescript
private async callLLM(systemPrompt: string, userMessage: string): Promise<string> {
  // Option 1: OpenAI
  const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
  const response = await openai.chat.completions.create({
    model: 'gpt-4',
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: userMessage }
    ],
    temperature: 0,
    response_format: { type: 'json_object' }
  });
  return response.choices[0].message.content || '';

  // Option 2: Anthropic
  const anthropic = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });
  const response = await anthropic.messages.create({
    model: 'claude-3-5-sonnet-20241022',
    max_tokens: 4096,
    system: systemPrompt,
    messages: [{ role: 'user', content: userMessage }]
  });
  return response.content[0].text;
}
```

## üìä Validation Coverage

### Errors (Blocking)
- ‚úÖ Route probabilities sum to 1.0 per station
- ‚úÖ Triangular distributions: `min ‚â§ mode ‚â§ max`
- ‚úÖ Normal distributions: `stdev > 0`
- ‚úÖ Exponential distributions: `mean > 0`
- ‚úÖ Uniform distributions: `min < max`
- ‚úÖ Station references exist
- ‚úÖ Entity references exist
- ‚úÖ At least 1 arrival policy defined
- ‚úÖ Run config: `warmup < runLength`
- ‚úÖ Counts and capacities ‚â• 1

### Warnings (Recommended)
- ‚ö†Ô∏è Disconnected stations
- ‚ö†Ô∏è Rework loops without escape
- ‚ö†Ô∏è High replication counts (performance)
- ‚ö†Ô∏è Class mix proportions sum to 1.0

### Auto-Fix Supported
- üîß Normalize route probabilities
- üîß Sort triangular distribution params
- üîß Swap min/max in uniform distributions
